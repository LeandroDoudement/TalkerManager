# Project Talker Manager

Esse repositório contém um projeto prático desenvolvido durante o curso de Desenvolvimento de Software da [Trybe](https://www.betrybe.com/). O objetivo do projeto é desenvolver uma aplicação para gerenciamento de projetos e comunicação entre membros de equipe.

## Descrição Geral

O Project Talker Manager é uma aplicação web para gerenciamento de projetos e comunicação entre membros de equipe. A aplicação possui funcionalidades como criação de projetos, cadastro de membros de equipe, adição de tarefas a projetos, atribuição de tarefas a membros de equipe, e troca de mensagens entre membros de equipe.

## Funcionalidades

O Project Talker Manager possui as seguintes funcionalidades:

- Cadastro de usuários
- Autenticação de usuários
- Criação de projetos
- Edição de projetos
- Remoção de projetos
- Adição de membros de equipe a projetos
- Remoção de membros de equipe de projetos
- Criação de tarefas em projetos
- Edição de tarefas em projetos
- Remoção de tarefas em projetos
- Atribuição de tarefas a membros de equipe
- Troca de mensagens entre membros de equipe

## Tecnologias utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- [Node.js](https://nodejs.org)
- [Express](https://expressjs.com/)
- [MongoDB](https://www.mongodb.com/)
- [Mongoose](https://mongoosejs.com/)
- [JWT](https://jwt.io/)
- [Socket.io](https://socket.io/)
- [Mocha](https://mochajs.org/) e [Chai](https://www.chaijs.com/) para testes automatizados
- [Swagger](https://swagger.io/) para documentação da API
